# 📱 Flutter Quiz App

Um aplicativo de quiz desenvolvido em Flutter, com o objetivo de testar conhecimentos de forma interativa e dinâmica. O app apresenta perguntas, permite selecionar respostas e exibe o resultado final ao usuário.

---

## 🚀 Funcionalidades

* ✅ Exibição de perguntas
* ✅ Múltiplas opções de resposta
* ✅ Seleção de alternativa
* ✅ Contagem de pontuação
* ✅ Tela de resultado final
* ✅ Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* Flutter
* Dart

---

## ▶️ Como executar o projeto

### 1. Clone o repositório

```
git clone https://github.com/GabriellyPiffer/Flutter_quiz.git
```

### 2. Acesse a pasta

```
cd flutter_quiz
```

### 3. Instale as dependências

```
flutter pub get
```

### 4. Execute o app

```
flutter run
```

---

## 📦 Gerar APK

Para gerar o APK do projeto:

```
flutter build apk --release
```

O arquivo será gerado em:

```
build/app/outputs/flutter-apk/app-release.apk
```

---

## 📱 Instalação no celular

1. Transfira o arquivo `.apk` para seu celular
2. Permita instalação de fontes desconhecidas
3. Abra o arquivo e instale

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com fins educacionais, visando praticar conceitos de desenvolvimento mobile com Flutter, como:

* Estrutura de widgets
* Gerenciamento de estado
* Navegação entre telas
